## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- JavaScript
- React
- API
- Git e Github

## 💻 Projeto

O GitFind é uma aplicação que busca e encontra dados no Github, seja de usuários, repositórios, projetos e afins, consumindo a API do github.
